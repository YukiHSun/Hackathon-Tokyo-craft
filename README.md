# Hackathon-Tokyo-craft

This project is part of the [Gemini 3 Tokyo Hackathon](https://cerebralvalley.ai/e/gemini-3-tokyo-hackathon).

## Status / ステータス
🚧 **Work in Progress / 作成中**

## Overview / 概要
Hackathon-Tokyo-craft is a project focused on building innovative solutions using Gemini.
Geminiを活用した革新的なソリューションを構築するためのプロジェクトです。

## Setup / セットアップ
1. Clone the repository. / リポジトリをクローンします。
2. Create a `.env` file and add your `GEMINI_API_KEY`. / `.env` ファイルを作成し、`GEMINI_API_KEY` を追加します。
3. Install dependencies: `pip install google-generativeai python-dotenv`. / 依存関係をインストールします： `pip install google-generativeai python-dotenv`
